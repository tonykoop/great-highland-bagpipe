# Resources

## System Context

The Great Highland Bagpipe packet is a system packet: chanter, drones, reeds,
bag, stocks, blowpipe, pressure regulation, and maintenance all interact. Public
claims should distinguish formula-derived dimensions from reed-and-pressure
validated dimensions.

## Sources To Confirm Before L3

- Modern pitch target and chanter scale reference used for the build.
- Commercial reed dimensions and pressure behavior.
- Drone reed seat and tuning-slide dimensions from measured examples.
- Bag, stock, and blowpipe interface dimensions from chosen hardware.
- Maintenance references for hemping, moisture control, and leakage checks.

## Related Repos

- `tonykoop/instrument-maker` for conical bore, stopped-pipe, and validation
  gates.
- `tonykoop/flutes` for tuner-log discipline. NAF K2 corrections do not apply
  to reed-driven bagpipe bores.
