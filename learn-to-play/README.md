# learn-to-play / Great Highland Bagpipe

Starter songbook for the Great Highland Bagpipe build packet. Generated by the sheet-music skill.

The bagpipe chanter uses a narrow nine-note scale over a constant drone. These arrangements keep the beginner focus on steady pressure, scale familiarity, and clean doublings.

Range: **Low G - High A on the Highland chanter. Written in pipe notation around A mixolydian.**

## Folder map
| Folder | Contents |

## Notes on notation
ABC files are learner sketches, not full pipe-band settings. Grace-note groups are marked plainly and should be replaced by house-style embellishments during instruction.

## L2 readiness and known gaps
This starter songbook is ready for L2 review as a playable packet draft: ABC melodies, per-tune notes, static fingering summaries, warmups, and a top-level chart are present. Known gaps: generated audio, MusicXML, LilyPond, MIDI, and printable songsheets are skipped until the shared render dependencies are available; fingering charts need builder review against the final hole layout.
